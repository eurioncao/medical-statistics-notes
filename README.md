# 📊 医学统计学笔记（Medical Statistics Notes）

🎓 帮助更多人从医学统计学的理解负担与考试压力中解放出来。

本仓库整理自《医学统计学（Medical Statistics）》课程内容，系统梳理核心概念、计算方法与常考题型，适用于医学、生物、公共卫生及相关专业的课程学习与复习参考。

📚 **主要参考教材**  
- 李康，贺佳 主编  
- 《医学统计学》  
- ISBN：978-7-117-36331-0
